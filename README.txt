# Biniyam Awalachew Firde - Portfolio Website

Welcome to my personal portfolio website! This website showcases my projects in data analysis, machine learning, and visualization.

Website URL:

biniyamafirde.github.io

# About the Website

This portfolio is built using a customized version of the Massively template by HTML5 UP. I have modified the template to feature my data-related projects, providing an overview of my work and skills in data cleaning, machine learning, and data visualization.

# Featured Projects:

Data Cleaning in SQL – Demonstrates data cleaning techniques using SQL to refine and structure raw datasets.

Data Cleaning with Python – Showcases the use of Python for cleaning and preprocessing data, leveraging Pandas and NumPy.

Tableau Projects – A collection of interactive Tableau dashboards for data visualization and insights.

Fishing Fleet Classification (Machine Learning) – A machine learning project analyzing and classifying fishing fleet data.

Cluster Analysis with Python – A clustering project using Python to identify patterns and group data points effectively.
